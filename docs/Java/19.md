# Kotlin

```kotlin
data class Response(
  val ci: String?,
) {
    
  init {
    ci = ci?:"hashed"
  }

  constructor(name: String, age: Int) : this(name) {
    println("Secondary constructor: Age is $age")
  }
}
```

#### init 블록 실행 순서

1. 객체 생성 및 필드 주입
2. init 블록 실행
   * 필드 값이 모두 주입된 후에, init 블록이 실행
   * init 블록은 생성자 호출이 끝난 직후 실행되며, 이 시점에서 이미 모든 파라미터가 설정되어 있는 상태
